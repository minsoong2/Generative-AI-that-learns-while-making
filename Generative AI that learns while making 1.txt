**생성 모델링(Generative Modeling)**
- 기계 학습과 인공 지능 분야에서 데이터를 생성하거나 시뮬레이션하는 모델
- 주어진 데이터셋을 학습, 새로운 데이터 생성
- ex) 사진, 음악, 텍스트 또는 음성, ...

생성 모델링의 종류

1. 생성적 적대 네트워크(GANs, Generative Adversarial Networks)
- 생성기(generator), 판별기(discriminator) 서로 경쟁하면서 학습
- 생성기(generator)
  - 진짜처럼 보이는 데이터를 생성함
- 판별기(discriminator)
  - 생성기가 생성한 진짜처럼 보이는 데이터와 원본 데이터를 구별함

2. 변분 오토인코더(VAEs, Variational Autoencoders)
- 입력 데이터를 압축
- 이후 원본 데이터와 유사한 형태로 복원하는 방식으로 작동
- 해당 과정을 통해 데이터의 핵심 특징 추출, 새로운 데이터 생성

3. 자기 회귀 모델(Autoregressive Model)
- sequence data(ex. txt, music)에 주로 사용
- 이전 데이터 포인트를 기반으로 다음 데이터 포인트를 예측하는 식으로 작동

**판별 모델링(Discriminative Modeling)**
- 기계 학습에서 입력 데이터를 특정 카테고리나 클래스로 분류하는 모델
- 주어진 입력 데이터가 어떤 레이블, 클래스에 속하는지 판단하기 위해 사용
- ex) spam email filtering, 의료 진단, 얼굴 인식, 언어 번역, ...

판별 모델링의 종류

1. 로지스틱 회귀(Logistic Regression)
- 이진 분류 문제에 주로 사용, 입력데이터가 특정 클래스에 속할 확률을 추정

2. 결정트리(Decision Tree)
- 데이터의 속성을 기반으로 결정을 내리는 브랜치 구조를 가진 모델
- 각 branch에서 데이터의 특징을 평가하여 최종결정

3. 서포트 벡터 머신(Support Vector Machine, SVMs)
- 데이터 포인트를 공간상에 매핑
- 서로 다른 클래스를 가장 잘 구분할 수 있는 경계를 찾음

4. 랜덤 포레스트(Random Forest)
- 여러 결정 트리를 조합하여 더 정확한 모델을 만듦
- 각 트리는 데이터셋의 서로 다른 부분을 학습
- 최종 결정은 트리들의 평균 or 다수결

5. 신경망(Neural Network)
- 심층 학습에서 사용되는 모델
- 입력 데이터에서의 패턴과 관계를 학습 -> 클래스 예측

**표현 학습 핵심 아이디어**
EX - 나의 외모를 모르면서 나를 찾고 있는 사람이 있다라 가정, 상대방이 평균적인 사람의 외모를 안다 가정
- 나는 흑발이다..., 안경을 쓰고 있다... 와 같이 픽셀 그룹에 해당하는 특성으로 이 가정을 개선
- 특성들이 여러개이면 픽셀로 다시 매핑하여 나의 이미지 생성
